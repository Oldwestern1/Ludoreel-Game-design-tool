# Ludoreel

A little game prompt tool for board/tabletop game design inspiration. Spin for a random mix of mechanics, themes, and components to spark a new game idea — lock the parts you like and reroll the rest.

## Features

- Spin across three categories — **mechanics**, **themes**, and **components** — each with adjustable pool size and per-item toggles, so you can narrow or expand what's in the mix.
- **Lock & reroll** individual results instead of starting over.
- **Prompt history** so you can step back through previous rolls.
- Custom pool items — add your own mechanics/themes/components alongside the built-in list.
- **"Learn more" links** on results — mechanics link to BoardGameGeek, evocative vocabulary words link to a dictionary definition, and most themes/components link to a relevant Wikipedia article.
- Light/dark theme and a handful of wallpaper styles.
- Sound effects (with a mute toggle), including a shared tick loop while cards spin.
- **Installable as a PWA** — add it to your home screen or desktop and it keeps working fully offline.

## Using it

Open the live page in a browser: https://oldwestern1.github.io/Ludoreel-Game-design-tool/, then install it as a PWA (look for "Add to Home Screen" / the install icon in your browser's address bar) after loading it once with a connection. (Chrome or Edge)
